# cashy

##Installation:

Zentral geht aus von cashshow.py aus

--> in webrequest.py muss die Google custom search engine eingerichtet werden um die API nutzen zu können
https://www.youtube.com/watch?v=ewPJs4N8d8M&list=WL&t=51s&index=56

--> für die Analyse der Antworten muss TreeTagger installiert werden
http://www.cis.uni-muenchen.de/~schmid/tools/TreeTagger/

--> für die Bildbearbeitung Tesseract installieren, inklusive aller Prerequisites
https://pypi.org/project/pytesseract/ 

--> ansonsten einfach mal alle Dateien durchgehen und die Paktete die importiert werden über pip installieren
